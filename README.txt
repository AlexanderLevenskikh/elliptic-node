Запуск:
После изменения файла input.json, запустить run.cmd. 
Результат в файле output.txt.

Формат файла input.json:
Содержит в себе пары ключ-значение. 
В случае char(F) > 3, необходимо указать module, a, b в виде строк, содержащих DEC/HEX число.
Умножение: указать factor, x, y (могут быть DEC/HEX)
Бесконечно удаленная точка - null в значении x и y (..."x": null, "y": null)
Сложение точек: указать x1, y1, x2, y2 (строк, содержащие DEC/HEX представления чисел)

Если char(F) === 2, вместо строк, содержащих числа, укажите массив показателей степеней ненулевых одночленов
x^4 + x + 1 <===> [ 4, 1, 0 ]
factor - строка, содержащая число в DEC/HEX представлении

для HEX-представления, необходимо перед числом добавить 0x (0x123)

Для каждого случая в текущей директории есть пример input-файла (файлы вида input_*.json)
